# Treinamento em Electron

Este repositório é dedicado ao **aprendizado e treinamento com Electron**, com foco na criação e experimentação de aplicações desktop utilizando tecnologias web.

O objetivo principal é praticar conceitos como **estruturação de projetos Electron**, integração entre **JavaScript, HTML e CSS**, manipulação de eventos e organização do código, por meio de exemplos, testes e pequenos projetos.

Este repositório funciona como um ambiente de estudo contínuo, testes de funcionalidades e consolidação dos conhecimentos adquiridos no desenvolvimento de aplicações desktop com Electron.
